## Packages Installation
1. Supabase Authentication and Implementation
``` terminal
npm install @supabase/supabase-js @supabase/auth-ui-react @supabase/auth-ui-shared
```

2. React Routing
``` terminal
npm install react-router-dom
```

3. Charts and Graphs
``` terminal
npm install recharts
```
